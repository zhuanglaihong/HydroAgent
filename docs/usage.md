# 使用指南

## 启动方式

### 交互模式

```bash
python -m hydroclaw
```

进入 REPL 界面，支持多轮对话。输入 `quit` 或 `Ctrl+C` 退出。

### 单次查询

```bash
python -m hydroclaw "你的查询内容"
```

执行单次任务后自动退出。

## 场景示例

### 1. 标准率定

```
You> 率定GR4J模型，流域12025000，使用SCE-UA算法，500轮迭代
```

HydroClaw 自动执行：验证流域 → 率定 → 评估 → 可视化 → 生成报告。

**指定更多参数**：

```
You> 率定XAJ模型，流域11532500，GA算法，种群大小50，训练期2000-2008，测试期2009-2014
```

### 2. 多模型对比

```
You> 用GR4J和XAJ两个模型分别率定流域12025000，对比哪个更好
```

LLM 会：
1. 分别对两个模型执行完整率定流程
2. 比较 NSE、RMSE 等指标
3. 给出推荐和分析

### 3. 多流域批量率定

```
You> 批量率定流域 12025000, 11532500, 12025000，使用GR4J模型
```

LLM 逐流域执行率定和评估，最后汇总对比。

### 4. 重复实验（稳定性分析）

```
You> 重复率定GR4J模型5次，流域12025000，分析稳定性
```

LLM 使用不同随机种子重复率定，计算 NSE 的均值、标准差、变异系数。

### 5. LLM 智能率定

```
You> 智能率定GR4J模型，流域12025000，目标NSE 0.75
```

流程：
- Round 1: 默认参数范围 → SCE-UA 优化 → 分析结果
- Round 2: LLM 调整参数范围（如扩展触界参数的边界）→ SCE-UA 优化
- 重复直到 NSE 达标或无法继续改进

### 6. 迭代优化（手动触发）

```
You> 率定GR4J模型，流域12025000。如果参数触碰边界，请扩展参数范围重新率定
```

LLM 读取 `iterative.md` Skill 后：
1. 第一轮率定
2. 检查参数是否距边界 <5%
3. 如触界 → 使用 `param_range_file` 扩展范围 → 重新率定
4. 如未触界但 NSE 低 → 换随机种子重试

### 7. 自定义分析

```
You> 率定完成后，帮我计算流域的径流系数，并画FDC曲线
```

LLM 调用 `generate_code` 生成分析脚本，再用 `run_code` 执行。

### 8. 自动创建工具

```
You> 帮我做参数敏感性分析，用SALib包
```

如果 `sensitivity_analysis` 工具不存在，LLM 会：
1. 调用 `create_tool` 生成工具脚本
2. 新工具自动注册
3. 调用新工具完成任务

## 语言支持

- 中文查询 → 中文回答
- 英文查询 → 英文回答
- 技术术语保留英文原文（NSE、RMSE 等）

## 输出说明

### 终端输出

```
============================================================
  User: 率定GR4J模型，流域12025000
============================================================

  >> Calling: validate_basin({"basin_ids": ["12025000"]})
  << validate_basin: valid=["12025000"]
  >> Calling: calibrate_model({...})
  << calibrate_model: {"NSE": 0.748, ...}

──────────────────────────────────────────────────────────────
  [LLM 的最终分析报告]
──────────────────────────────────────────────────────────────
```

### 文件输出

| 位置 | 内容 |
|------|------|
| `results/` | 率定结果（参数、指标、NetCDF） |
| `results/*/figures/` | 可视化图片 |
| `sessions/` | 会话 JSONL 日志 |
| `logs/` | 运行日志 |
| `generated_code/` | LLM 生成的分析脚本 |
| `MEMORY.md` | Agent 积累的跨会话知识 |

## 技巧

### 控制率定参数

```
You> 率定GR4J模型，流域12025000，SCE-UA算法，迭代只需100轮
```

### 指定数据时段

```
You> 率定GR4J模型，流域12025000，训练期1990-2000，测试期2001-2005
```

### 使用工作目录隔离实验

```bash
python -m hydroclaw -w results/experiment_A "率定GR4J..."
python -m hydroclaw -w results/experiment_B "率定XAJ..."
```
